# macOS admin snippets

This folder contains Python and zsh code snippets for common macOS administration tasks.

Files:
- `common_uses.py` / `common_uses.zsh`: logged in user, local accounts, installer, API call, JSON/YAML processing
- `files.py` / `files.zsh`: find, read, copy, edit, permissions, remove, create
- `advanced.py` / `advanced.zsh`: positional args, functions, error logging, prompts, loops
- `applied.py` / `applied.zsh`: LaunchAgents, Jamf EA, Munki-style install/uninstall scripts
